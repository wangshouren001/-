# 1、变量
## 1.1、变量必须先声明，再使用
如：
```
int x;  
x=1;
```
或  
```
int x=1;
```
而不能:
```
x = 1;
```
##### 其他类型同上，即凡是变量必须有且仅有一次申明
### 变量定义
```
type variable
```
或
```
type variable = value;
```
显然变量可赋值多次,也可不赋值
```
int x=1;
x=2;
x=3;
int z;
 .
 .
 .
```

### "声明" "定义" "赋值" 的内涵
> ### 声明：告知计算机一个变量的产生，通常形式是：”类型 变量名称”
> ### 定义： 比较广义，是解释一个概念，或者描述一个概念包含那些信息
> ### 赋值: 声明给变量一个名字(叫什么)，赋值给变量具体的内容(是什么、有什么、内容)
## 1.2、常量最多只能赋值一次  
### 常量的两种定义方式
#### a、普通常量   
```
const type variable;
```  
#### b、符号常量
##### 使用define关键字定义
```
#define name replaceValue  (注意后面没";")
```
如：
```
const float y = 3.14;
#define PI 3.14
#define COUNTRY "china"
```
而不能：
```
const float y = 3.14;
y = 3.15;
```
## 1.3、变量声明、赋值注意事项
### 常量声明、赋值不要远离第一次使用的地方，即使用时才声明、赋值，不要过度提前
好的方式
```
int x =0 ;
func1(x);
  .
  .
  .
```
坏的方式
```
int x =0 ;
  .
  .
  .
func1(x);
  .
  .
```

# 2、字符及字符串
## 字符声明同上
```
char c;
```

## 字符赋值有两种方式
### 数字
```
char c = 66;
```
### 字符
```
char c = 'B';
```
